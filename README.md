# weather-app

A simple weather app using a weather API.

For more information visit [The Odin Project](https://www.theodinproject.com/courses/javascript/lessons/weather-app).

## Built With

- HTML
- CSS
- JavaScript
- React

## API Used

- [OpenWeatherMap](https://openweathermap.org/)
- [Giphy](https://developers.giphy.com/)

## Extras Used

- CSS gradient by [uiGradients](https://uigradients.com/)
- Spinner by [Tobias Ahlin](https://tobiasahlin.com/spinkit/)
